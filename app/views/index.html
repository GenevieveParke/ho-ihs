About this service

The Immigration Health Surcharge (IHS) is a stand alone service that will be used by other UKVI applications.

The user has the option to add their existing IHS number or pay the health surcharge. The user is required to have an IHS number before they can make payment.

Some visas require the user to have an IHS even if they are exempt from the surcharge.

The screen flow which includes links to the development JIRAS.

Design contraints

This service has been designed with a time constraint and only the MVP will initially be delivered. It is for this reason that an email confirmation on the confirmation screen has not be included. A confirmation email will need to be sent if/when users are able to add different IHS numbers for different visa applications.

This design has not been user tested due to delivery time constraints.
